# YouTube-Trends-Intelligence-Dashboard-for-Advertising-Strategy
Automated Tableau dashboard analysing YouTube trending videos by category, region, and time to support advertising planning decisions.

This project delivers an automated **Tableau Public dashboard** designed for advertising planners to monitor YouTube video trends across categories, regions, and time.

The dashboard replaces manual weekly analysis with a self-service tool that supports daily decision-making for video advertising strategy.
---

## Business Objective
Enable marketing and planning teams to:
- Identify which video categories trend most frequently
- Compare trend patterns across regions
- Detect categories with exceptional popularity in the United States
- Monitor how trends evolve over time using a single source of truth

---

## Data
Source table:
- trending_by_time.csv

Fields:
- record_id: primary key  
- region: country / region  
- trending_date: date and time  
- category_title: video category  
- videos_count: number of trending videos  

Data is refreshed every 24 hours (UTC).

---

## Dashboard Content
The dashboard includes:
- Trending history by date and category (absolute values)

## Key Questions Answered
- Which video categories trend most frequently?
- How are trends distributed across regions?
- Which categories are especially popular in the United States?
- How do U.S. trends differ from other regions?
